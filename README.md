# Authorizator3000
Api-Gateway service based on Spring Gateway MVC, which provides easy IM management with popular IM-providers.  

## Key Features:
1. [ ] Basic User sign-up
2. [ ] Basic sign-in
3. [ ] OAuth2 sign-up and sign-in
4. [ ] OIDC sign-up and sign-in
5. [ ] LDAP sign-up and sign-in
6. [ ] User Authorization based on config
7. [ ] Routing based on config
8. [ ] CRUD REST API for User management
9. [ ] HTTP Session in Redis
10. [ ] Native Image
11. [ ] Observability
