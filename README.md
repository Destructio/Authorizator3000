# Authorizator3000
API Gateway based on Spring Gateway MVC and Spring Security with support of various IAM providers

## Key Features:
1. [x] Local user creation and management via REST
2. [x] Local sign-in
3. [x] OAuth2 sign-in
4. [x] OIDC sign-in
5. [x] Routing based on external config
6. [x] HTTP Session in Redis
