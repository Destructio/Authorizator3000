# Authorizator3000
API Gateway based on Spring Gateway MVC and Spring Security with support of various IAM providers

## Key Features:
1. [x] Basic User sign-up via REST API
2. [x] Basic sign-in
3. [x] OAuth2 sign-up and sign-in
4. [x] OIDC sign-up and sign-in
5. [ ] LDAP sign-up and sign-in
6. [ ] User Authorization based on config
7. [x] Routing based on config
8. [x] CRUD REST API for User management
9. [x] HTTP Session in Redis
10. [ ] Native Image
11. [ ] Observability
